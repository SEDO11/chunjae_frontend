2. 마인드 맵

최대 3번 까지만 a태그를 이용하는게 사용자 측면에서 좋음

web 페이지만 보여줄 수 있는 게 web 이다.

자바
2개 이상의 클래스를 하나의 java 파일안에 선언할 수 있지만 기본 클래스는 public 나머지는 접근제한자를 쓰지 말아야 한다.

오버로딩
하나의 클래스 내부에서 같은 이름으로 되있지만 매개변수의 개수, 타입이 달라 여러개로 선언 된 생성자, 메소드

오버라이딩
부모로 부터 상속을 받아서 자식 클래스에서 다르게 사용

static, static final 이 걸린것은 상속이 안되고 오버라이딩은 불가하다.

scope
변수의 유효 범위 (전역변수, 지역변수)

DTO
데이터 전송용 오브젝트

VO
값 저장용 오브젝트

Controlor
사용자에게 받은 데이터를 제어

DAO
데이터 액세스 오브젝트, DB에 연결하여 데이터를 제어하는 오브젝트

여러 사용자들이 한꺼번에 데이터를 전송할 수 있도록 나눠서 만든다.

객체지향의 특징
캡슐화, 재사용성, 상속성, 정보은닉
