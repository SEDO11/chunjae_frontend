CSS 출력 및 박스 관련 속성

display, visibility, opacity, width, height, overflow, margin, padding, border

모든 속성의 값에 initial(초기값), inherit(부모요소로 부터 상속)

display(출력속성)

화면 출력 모드를 지정하는 속성

태그마다 화면 출력 모드가 block, inline, inline-block로 나뉨

inline 속성 요소: span, a, strong, em, ins
특징: 한 줄 안에 자동 줄바꿈이 될 때까지 배치되고 크기 지정을 할 수 없어
width, height, line-height, text-indent, text-align, padding-top, padding-bottom, margin-top, margin-bottom
등의 속성 지정 불가

block 속성 요소 : div, h1~h6, p, section, article, header, footer, ...
특징: 자동으로 다음줄에 배치가 되며, 크기와 관련된 속성을 지정할 수 있음

부여 가능한 속성의 값 : inline | block | inline-block | none

부여 가능한 속성의 값2 : contents, flex, grid, box, table,
inline-flex, inline-grid, inline-table, list-item, run-in,
table-caption, table-column, table-row, table-cell, table-header-group, table-footer-group, ...

visibility(가시속성)

보일지 보이지 않을지를 나타내는 속성

visible | hidden | collapse

opacity(불투명도속성)

불투명 정도를 나타내는 속성

값의 범위 : 0~1 까지의 실수 or 정수로 표시, 0은 완전투명, 1은 완전불투명
해당 요소의 글자와 배경색, 이미지를 포함한 모든 콘텐츠를 불투명 또는 투명하게 지정가능

width(너비속성), height(높이속성)

가로, 세로 크기를 지정하는 속성

속성의 값 : auto | length | %

length는 px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

블록요소 또는 인라인블록요소 등에만 크기지정이 가능

overflow(흘러넘침속성)

콘텐츠의 양이 지정한 크기(폭,높이)보다 더 클 경우 지정

visible(넘처도 표시) | hidden(넘치면 숨김) | scroll(넘치든 안넘치든 스크롤바 생성) | auto(넘치면 스크롤바 생성, 안 넘치면 생성안함 | 기본값)

세부 속성 : overflow-x(가로의 overflow만 지정) | overflow-y(세로의 overflow만 지정) |

margin(바깥여백)

현재 콘텐츠 영역 영역의 바깥 여백 여러 콘텐츠와 콘텐츠 사이의 여백

속성의 값 : auto | length | %

length는 px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

값 입력 : margin: 상하 좌우 | 상 우 하 좌

속성 : margin-top, margin-bottom, margin-left, margin-right

주의사항 : 전체 속성을 먼저 지정 후 세부속성을 지정해야 한다.

padding(안쪽여백)

테두리와 콘텐츠 사이의 여백

속성의 값 : auto | length | %

length는 px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

값 입력 : padding: 상하 좌우 | 상 우 하 좌

속성 : padding-top, padding-bottom, padding-left, padding-right

주의사항 : 전체 속성을 먼저 지정 후 세부속성을 지정해야 하고
지정한 가로, 세로 크기에 padding 과 border를 더한 값이 실제 콘텐츠 크기가 된다.

border(테두리)

border : width style color

ex> : 1px solid #000 1픽셀 두께의 검은색 실선

width는 px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

style : 선의 모양으로서 solid(기본값, 실선), none(배정x) | hidden(배정o, 안보임)
| dotted(둥근 점선) | dashed | double(두줄 실선) | groove | ridge | inset | outset |

color 컬러명 | hex코드 | rgb/rgba, hsl/hsla 등으로 지정 가능


세부속성

border-width(선의 두께)

border-width : 2px
border-width : 2px 5px
border-width : 2px 5px 8px 10px

border-style(선 종류)

border-style : solid
border-style : solid dashed
border-style : solid dashed double hidden

border-color(선 색상)

border-color : red
border-color : red blue
border-color : blue gold navy orange

방향별 세부 속성

왼쪽 테두리
border-left : 2px double silver

오른쪽 테두리
border-right : 2px double silver

위쪽 테두리
border-top : 2px double silver

아래쪽 테두리
border-bottom : 2px double silver

방향별 세세 속성

왼쪽 테두리
border-left-width, border-left-style, border-left-color

오른쪽 테두리
border-right-width, border-right-style, border-right-color

위쪽 테두리
border-top-width, border-top-style, border-top-color

아래쪽 테두리
border-bottom-width, border-bottom-style, border-bottom-color

box sizing(박스의 크기 산정 방법)

box sizing은 박스의 크기를 지정한 너비/높이가 콘텐츠 까지만 할지, padding and border를 포함할 지정

content-box | border-box

content-box는 지정한 너비/높이가 콘텐츠에만 적용
border box는 지정한 너비/높이에 콘텐츠의 크기+padding+border 로 적용

border-radius(테두리둥글기)

속성의 값 : auto | length | %

length는 px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

값 입력 : padding: 상하 좌우 | 상 우 하 좌

세부속성

border-radius: 20px;
border-radius: 50%;
border-radius: 20px 50%;
border-radius: 10% 20% 30% 40%;

세세속성

border-top-left-radius: 20px;
border-top-right-radius: 50%;
border-bottom-left-radius: 20px;
border-bottom-right-radius: 50%;
