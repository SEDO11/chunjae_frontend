배치 및 레이아웃 관련 속성(중요)

position, left, right, top, bottom, z-index, float, clear 등



position(위치)

현재 선택된 요소를 어떠한 방식으로 배치할지를 정하는 위치 결정 방식

사용가능한 속성: static | relative | absolute | fiexed | sticky

모든 요소는 위에서 부터 아래로, 왼쪽에서 오른쪽으로 배치가 된다.


static : 정적 배치, position의 기본값

relative : 상대 배치, 내부 요소나 형제 요소와 상대적으로 배치할 경우 지정
absolute 인 내부 요소의 원점을 부모 요소로 하려고 할 경우 활용

static과 relative는 margin으로 위치를 조절하라 (left, right, top, bottom은 헷갈릴수 있음)
가로 방향으로 여러 요소를 배치할 경우 float 속성을 활용하여 배치

absolute : 절대 배치, x(가로)축의 left/right 속성과 y(세로)축의 top/bottom 속성으로 배치하고 가급적 margin은 사용하지 않는다.
요소들을 겹치기하여 배치할 경우 z-index(layer=층수) 속성을 지정하고, z-index는 숫자가 클 수록 위에 배치가 된다.

fixed : 고정 배치, 화면에 고정시키는 위치 결정법 x(가로)축의 left/right 속성과 y(세로)축의 top/bottom 속성으로 배치하고 가급적 margin은 사용하지 않는다.
요소들을 겹치기하여 배치할 경우 z-index(layer=층수) 속성을 지정하고, z-index는 숫자가 클 수록 위에 배치가 된다.

absolute와 fixed는 위치를 지정할 때 left/top or right/bottom 둘 중에 하나만 써라

위치값 지정

left : 왼쪽에서 부터의 위치

right : 오른쪽에서 부터의 위치

top : 위쪽에서 부터의 위치

bottom : 아래쪽에서 부터의 위치

지정가능한 키워드 : auto | length

length는 %, px, pt, pc, cm, mm, in, vw, vmin, vmax 등의 단위로 지정이 가능

auto는 현재 left 속성 값이 지정이 되어 있는 경우 right 속성 값을 변경하려고 할 경우 left를 auto로 하고, right 값을 지정하면 된다.



레이어 설정

z-index : layer(층수)의 위치로서 여러 요소를 겹칠 경우 어떤 요소를 앞으로 보내고, 뒤로 보낼 것인지 숫자 정수로 지정하여 결정하고, 숫자가 클수록 위층에 배치
기본 값 auto

만약, z-index 속성 값을 지정하지 않으면, 먼저 작성된

float - 흐름 속성

지정 가능한 값 : none | left | right

float : left; 를 가장 많이 사용

자식에게 float을 할당하면 부모요소는 높이가 자동으로 할당되지 않음
그래서 자동으로 맞추기 위해 after를 만들어 block 처리를 한다.

clear - 흐름 해제 속성

지정 가능한 값 : none(지정 안함) | left | right | both(둘다 해제)

clear : both; 이걸 많이 사용
